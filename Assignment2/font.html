Font Properties

When styling text on a webpage, there are several font properties you can adjust using CSS:

  • Font family
  • Font size
  • Font weight
  • Font style

Font Family

The font family property specifies the typeface for the text. There are several font families to choose from, such as:

  • Sans-serif fonts
    (e.g. Arial, Helvetica, Verdana)
  • Serif fonts
    (e.g. Times New Roman, Georgia)
  • Monospace fonts
    (e.g. Courier New, Lucida Console)
  • Cursive fonts
    (e.g. Comic Sans MS, Brush Script MT)
  • Fantasy fonts
    (e.g. Impact, Jokerman)

Font Size

The font size property specifies the size of the text. You can set the font size in pixels, ems, or as a percentage of the parent element's font size.
