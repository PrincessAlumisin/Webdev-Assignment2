Display Property

Display: Inline

When an element's display property is set to inline, the element will not begin on a new line and will occupy the entire amount of available screen space. It just occupies the space that such an element would typically occupy. This is an example of inline element.

Display: Block

When the display property is set to block, an element begins on a new line and takes up the entire available screen width.This is an example of a block element.

Display: Inline-Block

An element that you give an inline-block display to is inline by presentation. However, it has the added benefit of allowing you to apply width and height, which you are unable to do when the element has an inline display given to it. This is an example of inline-block element.

Display: None

When you set an element's display property to none, the element completely disappears from the page and has no bearing on the layout. This is an example of none element.

Display: Flex

A display of flex gives you access to the Flex layout system, which simplifies how we design and layout our web pages.This is an example flex display property

Display: Inherit

This makes the element inherit the display property of its parent. So, if you have a span tag inside a div and you give the span tag a display of inherit, it turns it from inline to block element.This is an example of inherit display property.

Display: Initial

This sets the display property of an element to its default value. So, if you set the display property of a span to initial, it remains inline, and if you set the same value for a div, it remains block.This is an example of initial display property

Display: Flex

A display of flex gives you access to the Flex layout system, which simplifies how we design and layout our web pages. This is an example of flex display property.
